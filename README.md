# TypeScript Template

A barebones TypeScript starter project.

## Setup

```bash
npm install
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run build` | Compile TypeScript to JavaScript in the `dist/` directory |
| `npm start` | Run the compiled output from `dist/` |
| `npm run dev` | Run `src/index.ts` directly using `tsx` (no build step needed) |
| `npm run lint` | Check for linting errors with ESLint |
| `npm run lint:fix` | Auto-fix linting errors |
| `npm run format` | Format all source files with Prettier |
